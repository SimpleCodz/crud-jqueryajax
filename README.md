# crud-jqueryajax
Aplikasi CRUD sederhana PHP MySQLi dengan AJAX JQuery

Untuk menginstall aplikasi ini dibutuhkan PHP versi 7 dengan database MySQL, dan dibuat dengan query MySQLi.
